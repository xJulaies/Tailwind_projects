Sudoku
_Board_

- 9x9 square field
  -> 9x9 field within each square
  ->numbers 1-9 in tiny notes within each tiny square

_logic_

- Numbers 1-9 in each square, row and column
  -> can only appear once!
- generate a field
- multiple difficulty settings
  -> hide certain amounts of numbers at game start

- Gameplay:
  - choose between normal mode and note mode
  - click a field to select/highlight it
  - choose a number
  - compare user number with field number
    -> if correct, fill field, otherwise color number red
